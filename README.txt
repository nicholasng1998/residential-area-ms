Application Start Up Requirements

Admin angular application:
1. Download NodeJs 16.13.0 using link below:
    a. https://nodejs.org/dist/v16.13.0/node-v16.13.0-x64.msi
2. Install Angular CLI using command below:
    a. npm install -g @angular/cli
3. Change to admin angular app directory using command below: 
    a. cd ./residential-area-admin
4. Install required angular packages using command below:
    a. npm install
5. Start the angular application using command below:
    a. npm run start

Backend-sevice application:
1. Download gradle 7.6 with link below
    a. https://gradle.org/next-steps/?version=7.6&format=bin
